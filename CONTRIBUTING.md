# 🤝 Contributing

Contributions, issues, and feature requests are welcome! If you are a fellow developer in the GATE community and want to improve this tool:

1. Fork the Project (and clone your fork locally).
2. Create your Feature Branch (`git checkout -b feature/AmazingFeature`).
3. Commit your Changes (`git commit -m 'Add some AmazingFeature'`).
4. Keep your branch updated and test: If the main project's dev branch has moved forward, pull those changes into your feature branch to test for conflicts (`git pull origin dev`).
5. Push the Feature Branch to your fork (`git push origin feature/AmazingFeature`).
6. Open a Pull Request: Go to the original repository and open a PR from your fork's `feature/AmazingFeature` branch into the original project's `dev` branch.
